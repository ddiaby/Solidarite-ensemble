# 🌍 Solidarité Ensemble

Un site web simple et responsive pour une **association** qui souhaite faire des **appels aux dons**.  
Ce projet a été généré à l’aide d’une **IA de code (ChatGPT)** dans le cadre de ma formation **Développeur Web et Web Mobile**.

---

## 🎯 Objectif du challenge / client

Projet réalisé pour le client fictif **PixelNova** afin d’apprendre à :
- Utiliser une IA générative pour produire du code (HTML / CSS)
- Comprendre la structure d’un site web
- Publier un projet complet sur GitHub

---

## 🧰 Technologies utilisées

- **HTML5**
- **CSS3**
- **IA générative (ChatGPT)**
- **Visual Studio Code**
- **Git & GitHub**
